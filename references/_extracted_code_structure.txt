二、 核心代码骨架设计 (PyTorch)
这里为你提供最核心的类的设计思路与骨架代码，你可以直接在这些骨架上填空。
1. 数据集加载 (data/dataset.py)
在这个任务中，每个样本不仅包含历史和未来轨迹，还需要包含对应的场景语义图和行人邻接矩阵（用于GAT）。
2. 个体轨迹编码器 (models/lstm_encoder.py)
使用简单的 LSTM 提取时间序列特征。
3. 场景语义特征提取 (models/resnet_seg.py)
这是课程的拿分重点。输入预处理好的语义分割图，利用预训练的 ResNet 提取高维特征，并降维以匹配轨迹特征。
4. 图注意力模块 (本科简化版 GAT) (models/gat.py)
无需使用复杂的图框架，用线性层和矩阵乘法实现基础的注意力交互即可。
5. 整体组装 (CSGAT-Net 主模型) (models/csgat_net.py)
将上述模块拼接，并加入 CVAE 的逻辑（训练时引入目标轨迹算均值方差，测试时随机采样）。

三、 训练与损失函数建议 (train.py)
对于 CVAE，损失函数必须包含两部分：轨迹重构损失 (L2/MSE) 和 KL散度 (约束隐变量分布)。
四、 针对课程论文拿高分的小建议
规避代码过于复杂： 本科论文重点考察逻辑完整性和思路理解。图注意力和 CVAE 的代码如果难以实现，可以用简单的多层感知机（MLP）和常规的 Encoder-Decoder 代替并解释为“计算资源的妥协与轻量化改进”，这反而能成为论文中的一个亮点（即“轻量级优化”）。
可视化（极其重要）： utils/visualize.py 一定要做好。写一个 Python 脚本，用 Matplotlib 将测试集中的语义分割图作为底图（imshow），然后用蓝色的点画历史轨迹，红色的点画真实轨迹，绿色的线画你模型预测的多模态轨迹。一张漂亮的可视化图在图像处理课程的答辩中能直接锁定高分。
